# FastApi

## Criando e ativando ambiente virtual

```bash
python3 -m venv .venv
source .venv/bin/activate
```

## Instalando dependências

```bash
pip install -r requirements.txt
```

## Rodando api

```bash
uvicorn api:app --reload
```

## Consultando

> Você pode acessar `localhost:8000/docs` para ver mais informações
